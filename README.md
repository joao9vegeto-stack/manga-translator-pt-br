# Manga Translator PT-BR

This repository is dedicated to translating manga into Brazilian Portuguese.

## Features
- Translate manga chapters.
- Community contributions for translations.
- Easy-to-use script for applying translations.

## Getting Started

### Prerequisites
- Python 3.x
- Required libraries (see `requirements.txt`)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/joao9vegeto-stack/manga-translator-pt-br.git
   ```
2. Navigate to the directory:
   ```bash
   cd manga-translator-pt-br
   ```
3. Install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```

### Usage
- To translate a chapter, run:
   ```bash
   python translate.py [chapter_name]
   ```
